Be aware:

  • Floats widthout width and the width of the content will be the width
  • Long text within float requires that you set a width. Otherwise it's 100%
  • A tall floated item must be cleared, or you will end up with floats within floats

Uncleared floats:

.float {
  float: right;
  margin-right: 20px;
  margin-bottom: 20px;
}

Uncleared floats:

Clear your floats:

  • No need to use extra markup to clear your floats. Self-clearing is the only way.
  • Clearfix mixin can help you, but better to use custom one

Self Clearing Floats:

.selector:after {
  content: "";
  display: block;
  height: 0;
  clear: both;
  visibility: hidden;
}

Positioning

Things to remember:

  • Absolutely positioned elements are positioned relative to the nearest relatively positioned element, or the body element if none.
  • Fixed and sticky positioned elements are positioned to the body element every time.

Absolute positioning:

.parent {
  position: relative;
}

.clild {
  position: absolute;
  top: 0;
  left: 0;
}

Absolute positioning

Fixed positioning:

.parent {
  position: relative;
}

.clild {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
}
